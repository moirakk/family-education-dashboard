# Family Education Dashboard

A production-ready MVP for multi-child household education management. The pilot experience ships with three children, while the architecture supports any family size and future SaaS commercialization.

## Stack

- Next.js 15 App Router
- TypeScript
- TailwindCSS
- shadcn/ui-style local primitives
- Supabase + PostgreSQL schema
- Vercel deployment target

## Local Development

```bash
npm install
npm run dev
```

Create `.env.local` from `.env.example` when wiring Supabase.
